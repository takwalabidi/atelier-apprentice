# 🪡 Atelier Apprentice - Maison Véline

Bienvenue sur le dépôt de **Maison Véline**, une application web moderne dédiée à une école de haute couture. Ce projet combine une interface élégante pour la présentation des formations et un système d'inscription pour les futurs élèves.

## 🌟 Aperçu
L'application permet aux passionnés de mode de découvrir des formations certifiantes à Paris et de s'y inscrire via un formulaire dynamique.

**Démo en ligne :** [atelier-apprentice.lovable.app](https://atelier-apprentice.lovable.app)

## 🚀 Technologies Utilisées

### Frontend
* **React + Vite** : Pour une interface rapide et réactive.
* **TypeScript** : Pour un code robuste et typé.
* **Tailwind CSS** : Pour un design épuré au style "Haute Couture".
* **Lucide React** : Pour les icônes.

### Backend (En cours de développement)
* **Python / Django** : Prévu pour la gestion des inscriptions et de la base de données des élèves.
* **Django Rest Framework** : Pour l'API de communication entre le front et le back.
